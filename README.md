# VOID PHANTOM 
### A Fast-Paced Roguelike Space Shooter | Open Source Edition

Void Phantom is a high-octane, pixel art roguelike space shooter. Navigate through hostile galaxies, customize your ship with deep crafting systems, and face massive biomechanical bosses. This project started as a technical experiment in 2024 and has evolved into a fully-fledged production title by Rpx.
## The Vision

This project is Source-Available. While the game is a commercial product on Steam, I have decided to open the codebase to the community to support fellow indie developers.

    For Developers: Use this as a real-world reference for roguelike architecture in Godot 4.

    For Players: Complete transparency on game balance, drop rates, and mechanics.

    The Goal: Excellence through transparency.

## Technical Stack

    Engine: Godot Engine 4.x (GDScript)

    Architecture: Component-based design for modular ship abilities.

    Backend: Silent Wolf (Online Leaderboards).

## Getting Started

    Clone the Repo: git clone https://github.com/Rp-ics/Void-Phantom

    Import to Godot: Open Godot 4.x and import project.godot.

    Dependencies: A valid Silent Wolf API key is required for leaderboard functionality.
